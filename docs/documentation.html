Quick Reference

Base URL

https://causalattribution.onrender.com

Authentication

All requests require an API key in the X-API-Key header:

X-API-Key: YOUR_API_KEY

Endpoints

1. Analyze Attribution

POST /api/v1/attribution

Perform multi-touch attribution analysis using causal inference.

{
  "touchpoints": [
    {
      "customer_id": "c001",
      "timestamp": "2025-10-01T10:00:00",
      "channel": "google_ads",
      "conversion": 0
    },
    {
      "customer_id": "c001",
      "timestamp": "2025-10-02T14:00:00",
      "channel": "email",
      "conversion": 1
    }
  ],
  "attribution_model": "data_driven",
  "conversion_window_days": 30
}

2. Calculate Campaign ROI

POST /api/v1/campaign-roi

Calculate ROI for marketing campaigns.

3. Optimize Budget Allocation

POST /api/v1/optimize-budget

Optimize budget distribution across channels.

4. Health Check

GET /health

Check API status (no authentication required).

5. Agent Schema

GET /api/v1/agent-schema

Get OpenAI/Anthropic function schemas for AI agents.

Attribution Models

  • data_driven - Causal inference using PC algorithm (recommended)
  • shapley - Game theory-based attribution
  • first_touch - 100% credit to first touchpoint
  • last_touch - 100% credit to last touchpoint
  • linear - Equal credit to all touchpoints
  • time_decay - More credit to recent touchpoints
  • position_based - 40% first, 40% last, 20% middle
