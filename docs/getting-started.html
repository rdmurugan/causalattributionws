Quick Start Guide

This guide will help you make your first API request to the Causal Attribution API in just a few minutes.

Step 1: Get Your API Key

To access the API, you'll need an API key. Getting one is simple:

  1. Email us at durai@infinidatum.net
  2. Include your company name and expected monthly touchpoint volume
  3. We'll respond within 24 hours with your API key and access details
  4. Start with our 14-day free trial - no credit card required

Note: During the trial period, you'll have access to all features of the Professional plan, including 250,000 touchpoints and 5,000 API calls per month.

Step 2: Authentication

The API uses API key authentication. Include your API key in the request headers:

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

Step 3: Make Your First Request

Here's a simple example using cURL to analyze attribution for a customer journey:

cURL Example

curl -X POST https://api.causalattribution.com/v1/attribute \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "touchpoints": [
      {
        "timestamp": "2025-01-01T10:00:00Z",
        "channel": "facebook_ad",
        "cost": 2.50,
        "user_id": "user123"
      },
      {
        "timestamp": "2025-01-02T15:30:00Z",
        "channel": "google_search",
        "cost": 3.75,
        "user_id": "user123"
      },
      {
        "timestamp": "2025-01-03T18:45:00Z",
        "channel": "email",
        "cost": 0.10,
        "user_id": "user123"
      }
    ],
    "conversion": {
      "timestamp": "2025-01-03T19:00:00Z",
      "value": 150.00,
      "user_id": "user123"
    },
    "model": "data_driven"
  }'

Step 4: Understanding the Response

The API returns a JSON response with attribution scores for each touchpoint:

{
  "attribution": {
    "facebook_ad": {
      "credit": 0.35,
      "attributed_value": 52.50,
      "roi": 21.00
    },
    "google_search": {
      "credit": 0.45,
      "attributed_value": 67.50,
      "roi": 18.00
    },
    "email": {
      "credit": 0.20,
      "attributed_value": 30.00,
      "roi": 300.00
    }
  },
  "model_used": "data_driven",
  "total_conversion_value": 150.00,
  "confidence_score": 0.92,
  "causal_graph": {
    "edges": [
      {"from": "facebook_ad", "to": "google_search", "strength": 0.65},
      {"from": "google_search", "to": "email", "strength": 0.42}
    ]
  }
}

Response Fields Explained

  • credit: The proportion of conversion credit (0-1) assigned to each touchpoint
  • attributed_value: The dollar value attributed to this touchpoint (credit √ó conversion value)
  • roi: Return on investment for this touchpoint (attributed_value / cost)
  • confidence_score: Statistical confidence in the attribution results (0-1)
  • causal_graph: Discovered causal relationships between touchpoints

Available Attribution Models

You can specify different attribution models in your request:

  • data_driven - Our proprietary causal inference model (recommended)
  • shapley - Shapley Value attribution (game theory-based)
  • first_touch - All credit to first touchpoint
  • last_touch - All credit to last touchpoint
  • linear - Equal credit to all touchpoints
  • time_decay - More credit to recent touchpoints
  • position_based - 40% first, 40% last, 20% middle
